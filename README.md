# BrightTV-Case-Study

BrightTV ‘s CEO has an objective to grow the company’s subscription base for this financial
year. He has approached you to provide insights that would assist CVM (Customer Value
Management) team in meeting this year’s objective.

The aim of the project to prepare a preseatation that will provide insights on user and veiwership trends on the BrightTV platform.

Questions to answer:
What factors influence consumption?
What content would you recommend to increase consumption on the days with low
consumption?
Are there any further initiatives to further grow BrightTV ‘s User base?

Tools used in the case study. 
- Databricks: Excel dataset is uploaded in databrucks for cleaning and processing
- Excel: Processed data will be exported to excel for deeper analysis, charting and graphing to generate insights
- Canva: Summary of resulta will be presented on slide deck.
- PowerBI Dashboard: For tracking of important metrics that serve as a summary of the CEO can use to get a feel of the companies health of the business.
